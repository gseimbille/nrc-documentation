## Liste des périmètres
Dans le menu de gauche de Datacake, sélectionnez **Perimeter** (ou **MSA FR**) pour afficher la liste des périmètres.

![Liste des périmètres dans Datacake](/img/clovir/image4.png)

## Création d'un périmètre
1. Cliquez sur **Créer un nouveau périmètre**.
2. Sélectionnez un tag existant dans Datacake (à créer au préalable si besoin).
3. Ajoutez une description (optionnel).
4. Cliquez sur **Ouvrir la carte** pour définir le périmètre dans l'interface geojson.io.
5. Sélectionnez l'outil **rectangle** ou **étoile** pour dessiner le polygone du périmètre.
6. Copiez-collez les coordonnées au format GeoJSON.
7. Définissez le nombre d'animaux attendus dans le périmètre.

![Création d'un périmètre](/img/clovir/image15.png)

![Exemple de paramètres](/img/clovir/image5.png)

:::caution
Seuls les périmètres en forme de quadrilatères (4 côtés) sont supportés en version 1.
:::
