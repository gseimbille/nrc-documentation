Vous pouvez consulter le tableau directement sur votre smartphone via l'application Datacake :
- iPhone : `https://apps.apple.com/us/app/datacake/id1358497267`
- Android : `https://play.google.com/store/apps/details?id=com.datacakehq.datacakedashboardapp`

Dans l'application :
1. Supprimez les devices de démo dans **Settings** > **Remove devices**.
2. Ajoutez un nouveau device en appuyant sur l'icône **+** de la page d'accueil.
3. Scannez le QR code généré lors du partage du tableau de bord.
