---
title: Tableau de bord d'un collier
sidebar_position: 5
---

Cliquez sur **Devices** dans le menu de gauche puis sélectionnez le device (collier) que vous souhaitez consulter.

Un collier est **online** (vert) lorsqu'un message provenant du collier a été reçu par l'application. Dans le cas contraire, **offline** est rouge et indique que le collier est hors service (éteint, n'émet pas de message ou est en zone blanche).

Un collier est **in geofence** lorsqu'il se trouve dans le périmètre autorisé.

À droite, en face des pictogrammes, vous trouvez dans l'ordre :
- Le nombre d'alarmes reçues (dépassements du périmètre autorisé)
- Le nombre de stimuli électriques générés par le collier
- Le nombre de messages de géolocalisation reçus par l'application
- La température extérieure exprimée en degrés Celsius
- La tension de batterie en Volts : 4,2 V (batterie chargée), 3,2 V (batterie à recharger)
