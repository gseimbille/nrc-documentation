## Démarrage
1. Dessinez un périmètre 0 en extérieur pour la mise sous tension des colliers (PC).
2. Mettez sous tension les colliers dans ce périmètre 0.
3. Assurez-vous que les colliers apparaissent **online** et **in zone GPS** (PC ou téléphone).
4. Sortez les colliers du périmètre 0 et vérifiez que les sons et stimuli électriques sont générés, et qu'un SMS est reçu pour le franchissement.

## Activation du périmètre
1. Dessinez le ou les périmètres autorisés (1, 2, 3, etc.) sur Datacake (PC).
2. Activez le périmètre choisi.
3. Mettez le collier sur la vache et amenez l'animal à l'intérieur du périmètre défini à l'étape 1.
4. Vérifiez les colliers **in zone** sur Datacake.
5. Le périmètre sélectionné est opérationnel.

## Changement de périmètre
1. Dessinez le nouveau périmètre (si déjà fait, passez à l'étape suivante).
2. Désactivez le périmètre opérationnel.
3. Amenez les animaux dans le nouveau périmètre.
4. Activez le nouveau périmètre.
5. Le périmètre sélectionné est opérationnel.
