## Perimeter list
In the left Datacake menu, select **Perimeter** (or **MSA FR**) to view the list of perimeters.

![Perimeter list in Datacake](/img/clovir/image4.png)

## Create a perimeter
1. Click **Create a new perimeter**.
2. Select an existing tag in Datacake (create it first if needed).
3. Add an optional description.
4. Click **Open map** to define the perimeter in the geojson.io interface.
5. Use the **rectangle** or **star** tool to draw the perimeter polygon.
6. Copy and paste the perimeter coordinates in GeoJSON format.
7. Define the expected number of animals in the perimeter.

![Create perimeter flow](/img/clovir/image15.png)

![Perimeter settings example](/img/clovir/image5.png)

:::caution
Only quadrilateral perimeters (4 sides) are supported in version 1.
:::
